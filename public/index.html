klyn

A collection of purely client-side single page apps. See also the GitHub repo.

Gallery

A dynamic image viewer that loads and displays images based on a JSON manifest file. Provides a thumbnail grid view and full-resolution image viewing.

Usage: gallery.html?from=<url>

JSON Viewer

A client-side JSON viewer that fetches and displays formatted JSON data from URLs.

Usage: json-viewer.html?from=<url>

Markedup

A client-side markdown viewer that fetches and renders GitHub-flavored markdown files from URLs using marked.js.

Usage: markedup.html?from=<url>

These are single-page apps that can be served as static files, with no server-side processing required.
